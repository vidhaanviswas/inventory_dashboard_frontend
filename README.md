## ğŸ§¾ About

The **Inventory Dashboard Frontend** is a React + Vite application that provides a comprehensive and intuitive interface for managing inventory data.

It aims to help businesses:

- Efficiently track stock levels
- Visualize inventory trends
- Monitor warehouse performance
- Support data-driven decision making

The target audience includes small to medium-sized businesses, retailers, and warehouse managers who need a clean, easy-to-use tool for day-to-day inventory operations.

The frontend consumes REST APIs from a separate backend service (Node.js + MongoDB Atlas) and focuses on:

- Modern UI
- Performance
- Maintainable, component-based architecture
